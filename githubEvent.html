The SCADE Club, under the Department of CSE (Cyber Security & Data Science), organized a seminar titled "GitHub Bootcamp: Beginner to Expert." The guest speaker for the session was “Ms. Sanika Kotigre”, a seasoned expert in utilizing GitHub. The event aimed to educate students about the practical uses of GitHub in professional environments, covering its role in version control, collaboration, and project management. Students also learned best practices for using GitHub in collaborative projects. The seminar provided hands-on exercises and real-world examples, enabling students to gain practical experience and enhance their skills. They provided detailed insights into various Git commands, as well as key processes such as Git merging and Git branching.

The event featured interactive Q&A sessions where students could seek clarification on specific Git and GitHub features. By the end of the seminar, students gained practical experience and a deeper understanding of how to leverage Git and GitHub tools to enhance their coding practices, manage projects efficiently, and contribute effectively to team-based development environments. Total 80 students attended and benefited from the session. The session was handled by Ms. Swati Gajarlewar as the Activity Coordinator under the guidance of Dr. Deepika Ajalkar HOD CS & DS, and Dr. R D Kharadkar, Campus Director, GHRCEM.
